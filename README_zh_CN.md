## 1 目的及背景

当我进入北大的物理学院以后，我发现我对自己所学的专业并不感兴趣。在逐步探索的过程中，我发现自己对计算机科学真正感兴趣，便踏上自学相关课程的道路，希望以后能在相关领域做出自己的一点贡献。然而，当我开始真正自学时，我发现困难重重，不仅难以寻找合适的资源，而且想要系统的学习也并非容易之事。多亏一位学长向我推荐了[CS自学指南](https://csdiy.wiki/)，我终于找到了前进的正确道路。

一天我在学习时，突然想到：“嘿，如果能把我所有自学的笔记整合起来，并把它发布在自己的网页上，这样其他人在学习计算机科学时，可以参考我的笔记，这样多好啊。”于是，这份笔记应运而生。如今这份笔记已经发布至我的网站上，它也许没有那么准确完美，但是希望可以在您学习相关课程时给予您帮助。

## 2 使用建议

1. 本笔记的作者非计算机科学专业学生，笔记中难免有错误与疏漏，请您谅解。

2. 本文使用部分维基百科的图片，出于知识产权考虑并未对其进行修改，但是其中部分图片没有背景，在夜间模式下阅读效果可能不佳。笔记默认为夜间模式，建议您点击右上角最左侧按钮切换至日间模式。

3. 本笔记在最终结束后会发布pdf版本，您将在release部分看到。

4. 感谢您的阅读！如果有任何建议，欢迎通过邮箱联系我。

## 4 致谢

首先，我要向以上列表中所有教授、助教等表示感谢，他们不仅讲课生动形象，同时还将大量的学习资料面向公众开放，我从他们身上学到很多知识。

其次，我要向在自学过程中给予我帮助的师长、学长表示感谢。

最后，我想把这部分留给我的母亲。在过去的岁月里，她倾尽心血养育我，为我不辞辛劳地付出。我们经历了无数的风风雨雨；在如此艰难的环境下养育我极其困难，然而她依然无怨无悔地为我付出。在我自学计算机科学相关课程的过程中，无论我遇到多么大的困难，她都在背后默默的支持我、鼓励我。这个小小的项目显然不能称作为一种“成功”，但是我想把我所有的小成就都归功于她；没有她，我将不可能有任何成就。
