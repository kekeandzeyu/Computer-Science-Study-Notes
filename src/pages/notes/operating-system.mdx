## 1 Operating System Overview

An operating system (OS) is system software that manages computer hardware and software 
resources, and provides common services for computer programs.

Operating systems have many functions: 

1. Glue: Provides a set of common services.
2. Illusionist: Provide clean, easy-to-use abstractions of physical resources.
3. Referee: Manage protection, isolation, and sharing of resources.
